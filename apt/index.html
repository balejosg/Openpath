OpenPath System APT Repository

Quick Install

curl -fsSL https://balejosg.github.io/openpath/apt/apt-bootstrap.sh | sudo bash

Manual install (advanced):

curl -fsSL https://balejosg.github.io/openpath/apt/apt-setup.sh | sudo bash
sudo apt install openpath-dnsmasq
sudo openpath setup

Files

  • apt-bootstrap.sh - One-liner install + classroom setup
  • apt-setup.sh - Setup script
  • pubkey.gpg - GPG public key
  • dists/ - Distribution metadata
  • pool/ - Package pool
