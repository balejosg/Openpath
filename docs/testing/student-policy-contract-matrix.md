# Student Policy Contract Matrix

> Status: maintained
> Applies to: OpenPath student-policy verification
> Last verified: 2026-04-29
> Source of truth: `docs/testing/student-policy-contract-matrix.md`

This matrix maps the Selenium student-policy scenario families to the cheapest
maintained local contracts and the runner evidence that still proves real
browser, DNS, firewall, native-host, and policy propagation behavior. Local
contracts should stay seconds-scale and must not replace Linux or Windows
student-policy runner gates.

## Maintenance Rules

- Add or update a row whenever `tests/selenium/student-policy-scenarios.ts`
  gains a new `SP-*` or `SP-FB-*` scenario id.
- Local contract coverage must name concrete test files, not broad workspaces.
- Runner-only evidence must state what cannot be proven without the target
  platform.
- Keep recommended commands narrow enough for development-time hypothesis
  checks.
